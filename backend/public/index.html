Documentación de la API BookTags

Descripción

Esta es una API RESTful para la gestión de libros y autores.

Endpoints disponibles

1. Registrar Usuario

Método: POST

Ruta: /auth/register

Descripción: Permite registrar un nuevo usuario.


POST http://localhost:3000/auth/register

Headers:
Content-Type: application/json

Body (JSON):
{
    "email": "emailexample",
    "password": "passwordexample"
}

2. Iniciar Sesión

Método: POST

Ruta: /auth/login

Descripción: Inicia sesión y devuelve un token JWT.


POST http://localhost:3000/auth/login

Headers:
Content-Type: application/json

Body (JSON):
{
    "email": "emailexample",
    "password": "passwordexample"
}

3. Crear un Autor

Método: POST

Ruta: /author

Descripción: Crea un nuevo autor.


POST http://localhost:3000/author/ 

Headers:
Authorization: Bearer {token}
Content-Type: application/json

Body (JSON):
{
    "name": "Stephen King",
    "dob": "21/10/1947",
    "description": "Stephen Edwin King, más conocido como Stephen King y ocasionalmente por su pseudónimo Richard Bachman, es un escritor estadounidense de novelas de terror, ficción sobrenatural, misterio, ciencia ficción y literatura fantástica."
}

4. Obtener Todos los Autores

Método: GET

Ruta: /authors

Descripción: Obtiene una lista de todos los autores y sus ids, los cuales deben ser usados en la creación de libros


GET http://localhost:3000/author/

Headers:
Authorization: Bearer {token}

5. Obtener Todos los Libros

Método: GET

Ruta: /libros

Descripción: Obtiene una lista de todos los libros.


GET http://localhost:3000/libros

Headers:
Authorization: Bearer {token}

6. Filtrar Libros

Método: GET

Ruta: /libros/filter

Descripción: Filtra los libros por (title, yearPublished, genre), devuelve los libros que coincidan con estos parametros


GET http://localhost:3000/libros/filter?param=value

Headers:
Authorization: Bearer {token}

7. Obtener Libro por ID

Método: GET

Ruta: /libros/:id

Descripción: Obtiene un libro por su ID.


GET http://localhost:3000/libros/:id

Headers:
Authorization: Bearer {token}

8. Crear un Libro

Método: POST

Ruta: /libros

Descripción: Crea un nuevo libro, todos los campos son obligatorios y se debe elegir un género.


POST http://localhost:3000/libros

Headers:
Authorization: Bearer {token}
Content-Type: application/json

Body (JSON):
{
    "title": String,
    "author": (Stephen king ID),
    "yearPublished": Number,
    "genre": {
        'Ficción',
        'No ficción',
        'Ciencia ficción',
        'Fantasía',
        'Misterio',
        'Biografía',
    },
    "pages": Number
}

9. Actualizar un Libro

Método: PUT

Ruta: /libros/:id

Descripción: Actualiza la información de un libro existente.


PUT http://localhost:3000/libros/:id

Headers:
Authorization: Bearer {token}
Content-Type: application/json

Body (JSON):
{
    "title": "Título Actualizado",
    "author": "Autor Actualizado"
}

10. Eliminar un Libro

Método: DELETE

Ruta: /libros/:id

Descripción: Elimina un libro por su ID.


DELETE http://localhost:3000/libros/{id}

Headers:
Authorization: Bearer {token}

11. Obtener libro por title

Método: GET

Ruta: /libros/title/:tile

Descripción:Obtener un libro por su titulo


GET http://localhost:3000/libros/title/:title

Headers:
Authorization: Bearer {token}

12. Obtener Libros Ordenados

Método: GET

Ruta: /libros/sort

Descripción: Obtiene una lista de libros ordenados por el campo especificado en los parámetros de consulta.


GET http://localhost:3000/libros/sort?sort={campo}&order={asc|desc}

Headers:
Authorization: Bearer {token}

Parámetros de Consulta:
- sort: Campo por el cual ordenar (por defecto: title). Ejemplo: title, yearPublished, genre, pages.
- order: Dirección del orden (por defecto: asc). Valores posibles: asc o desc.
